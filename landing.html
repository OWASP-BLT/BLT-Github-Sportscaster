BLT GitHub Sportscaster

Real-time GitHub event intelligence with a live sportscaster feed, ranked activity leaderboard, AI commentary, and practical controls for monitoring the repos and communities that matter to you.

Watch Live Dashboard View Source

What it does

Live Event Stream

Tracks GitHub public activity in real-time with configurable scope for global, org, repo, or user views.

Ranked Leaderboard

Weighted activity scoring surfaces trending repositories and contributor momentum as it happens.

AI Commentary

Pluggable OpenAI-compatible narration adds play-by-play context with fallback templates when AI is unavailable.

Channel Filters

Quickly focus on hot, code, social, or all activity channels for cleaner monitoring during triage and demos.

Rate-Safe Polling

Countdown, progress bar, and auto-protect throttling help stay within GitHub API limits while remaining responsive.

Audio Controls

Independent sound-effects and text-to-speech toggles make it usable in both ambient screens and quiet workflows.

Why it helps

Teams get a single live board for engineering momentum, community signals, and release cadence.

How to use

Pick scope and filters in Settings, tune refresh speed, then watch leaderboard shifts and commentary.

Where it fits

Works well for demo walls, hackathons, community ops, and rapid OSS ecosystem monitoring.
